## Content Rules (Always Apply)

1. Never use generic phrases like "in today's world" or "as we all know"
2. Always include at least **one specific Nagpur reference** per post (landmark, corridor, locality)
3. WhatsApp posts must feel like a **genuine message from a trusted friend**, never a sales pitch
4. Every post must end with a **clear CTA** (call / DM / site visit / WhatsApp)
5. **RERA number must be mentioned** in every major project post
6. **No guaranteed return claims.** Replace with: "historical data shows," "based on NMRDA corridor growth"
7. Never mention competitor brand names. Refer to them as "some developers" or "others in [corridor]"
8. Prioritize **long-term value content** (educational, market insights) over hard-sell
9. **Sensitive data auto-redaction:** Names, phone numbers, exact addresses require explicit approval before publishing
