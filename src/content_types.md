# Content Types and Platform Strategies

| Platform      | Content Type    | Primary Goal                           |
| ------------- | --------------- | -------------------------------------- |
| **Instagram** | Video Reels     | Reach new people (discovery)           |
| Instagram     | Carousel Images | Engagement + saves (trust building)    |
| Instagram     | Single Image    | Listings (quick posts)                 |
| Instagram     | Text only       | Almost useless on Instagram            |
| **WhatsApp**  | Text            | King of personal, direct communication |

**Usage Note**: Store this file as a reference for content creation workflows. It can be imported by scripts or used by the team when planning social media calendars.
